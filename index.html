Hello!

This is the Sara Greer who lives in South Texas, works in IT, and is currently hiding from coronavirus with her husband, 2 dogs, and a cat. After several years of waiting for the domain to become available, I bought saragreer.com in May 2020. And here it sits while I decide what to do with it.

So far I've:

  • Bought the domain!
  • Hosted from a GitHub repository using GitHub Pages. It's free!
  • Managed my custom domain with the help of a free Cloudflare account. I'm going to be honest, it made a little sense while I was setting it up, but now I couldn't do it again without this how-to.
  • Wrote this quick and dirty intro, and threw a custom font up for a little pizzazz. I've been collecting Creative Market freebies for years, and have built up quite the cache of font faces. This one is called Neretto Sans.
  • Set up an email-to-post system so I can quickly share notes and links!
